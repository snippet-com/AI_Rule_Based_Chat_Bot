# AI_Rule_Based_Chat_Bot

A simple beginner-friendly AI chatbot developed using Python. This chatbot uses predefined rules and `if-else` decision-making logic to respond to user inputs.

The project demonstrates basic Artificial Intelligence concepts without using machine learning or NLP libraries.

---

## Features

- Handles greetings like "hi", "hello", and "hey"
- Responds to predefined questions
- Handles exit commands
- Runs continuously using loops
- Case-insensitive user input handling
- Simple and easy-to-understand code structure
